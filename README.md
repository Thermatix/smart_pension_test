# Specs

run `ruby lib/specs.rb` to run specs

# Usage

```
parse log_file_name.log [-h ] [ -m | -u ] [ -i ]
       -h, --help:   display help
       -m, --most:   page views, most viewed to least viewed
       -u, --unique: unique page views, most_viewed to least_viewed
       -i, --invese: inverse sort
```
